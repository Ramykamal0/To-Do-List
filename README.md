# 📝 To-Do List

A simple console-based To-Do List application built with C# to help users manage their daily tasks.

## 🚀 Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Simple and user-friendly console interface

## 🛠️ Technologies Used

- C#
- .NET
- Object-Oriented Programming (OOP)

## 📂 Project Structure

```
To-Do-List/
│
├── Program.cs
├── Task.cs
├── TaskManager.cs
└── README.md
```

## 📸 Example

```text
1. Add Task
2. View Tasks
3. Complete Task
4. Delete Task
5. Exit

Choose an option:
```
